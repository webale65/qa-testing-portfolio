# 📦 Project

**RiderFit – Precision E-commerce for Equestrian Gear**

RiderFit is an intelligent assistant integrated into an e-commerce platform that helps riders choose the correct equipment size by analyzing both **rider measurements** and **horse measurements**.

The system combines user data with product specifications to generate accurate recommendations.

# 🧪 QA Activities Performed

During the project development the following QA activities were performed:

---

# 🔎 Key Findings

Some relevant findings detected during QA validation include:

* Potential duplication of measurements during PUT operation in the RiderForm module.
* Hardcoded measurementTypeId values in frontend scripts creating risk of backend misalignment.
* Endpoints executing logic without consistent JWT validation.

These findings were documented and shared with the development team to improve system robustness.
